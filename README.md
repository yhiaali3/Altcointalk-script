# Altcointalks Extension

Extension customized for Altcointalks - Synthesis script, all in one.

Altcointalks forum is targeted by this adaptation; the project adapts features for that community.
This extension summarizes the features that members prefer, making it easier for users to use the Altcointalks website.

# Info:

* Open source: https://github.com/yhiaali3/altcoinstalks-script
* Can be installed on all browsers with Chromium core: Chrome, Brave, Microsoft Edge...
* For Firefox: coming soon....

  Latest version: 1.0 on 2-1-2026

  # Features:

* Change the theme
* Block the signature
* Block avatar
* Change the font size
* Pin multiple post or comment
* Show icon go to top (Always on)
* Enhanced Report To Moderator UI (Always on)
* Show Bitcoin \& Ethereum price + Option to add Altcoin as desired
* Themes Manager: Added full manager: Save, Apply, Delete, Edite, Export, Import
* Modify themes, delete some, and add new ones.
* Add a theme specifically for colorblind users.
* Add an "Edit" button to modify the code and name of any saved theme.
* Add an image upload button that uploads images to a hosting site and retrieves a BBcode link with the ability to resize them.
* Add an advanced Quill editor with features such as bold, zoom, colors, horizontal font, quote, code, and alignment directly
  within the editor, then convert it to BBcode.
